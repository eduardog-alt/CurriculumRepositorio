Experimentos Técnicos

Monitor de atividade

Ferramenta que coleta informações do sistema de usuários em máquinas corporativas

▼

Sistema experimental de monitoramento de atividade desenvolvido para estudar coleta confiável de dados em ambientes corporativos. A arquitetura é dividida em dois componentes: um agente executado como serviço do sistema responsável pela validação e envio das informações para uma API, e um helper executado no contexto do usuário responsável pela coleta de dados da sessão ativa. A comunicação entre os dois processos ocorre através de Named Pipes, permitindo que informações coletadas no nível do usuário sejam encaminhadas para o agente com privilégios elevados de forma controlada. O projeto explora conceitos de separação de privilégios, monitoramento de sistemas e comunicação entre processos, buscando reduzir a possibilidade de manipulação direta das informações pelo usuário.

C# Windows Service SQLite Privilege Separation API Segurança Background Service

Assistente virtual para jogos

Assistente que interage com jogos em emuladores.

▼

BMO é um assistente virtual experimental criado para interagir com jogos rodando em emuladores. O sistema utiliza uma wake word para ativação e analisa o estado do jogo através da leitura de memória exposta pela interface Libretro. A partir dessas informações o assistente consegue identificar eventos do jogo, responder perguntas sobre o personagem e executar ações como abrir jogos automaticamente no emulador. O projeto explora conceitos de inspeção de memória, integração entre processos e automação de jogos, buscando criar um "companheiro de gameplay" capaz de compreender o estado do jogo em tempo real.
